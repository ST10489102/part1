Cybersecurity Awareness Bot — Part 1

A C# console chatbot that educates South African citizens about cybersecurity threats and how to stay safe online.

 How to Run
- Open the project in Visual Studio
- Press F5 to run
- Enter your name when prompted
- Type a topic to get cybersecurity advice
- Type 'help' to see all available topics
- Type 'exit' to quit

Features
- Voice greeting using text to speech
- ASCII art logo
- Personalised user interaction
- 10 cybersecurity topics
- Input validation
- Typing effect and purple coloured UI

Project Structure
- Program.cs — entry point of the application
- ChatBot.cs — chat loop and voice greeting
- Display.cs — visuals, typing effect and formatting
- Responses.cs — all cybersecurity responses
